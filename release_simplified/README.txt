## 🎉 v3.1 增强更新
- 🔍 全新智能卖家信息提取算法
- 📞 精准电话号码识别和提取
- 🏢 完整公司地址和代表人信息

## 🚀 使用方法
1. 双击运行程序
2. 输入搜索关键词（如：电脑、笔记本、手机）
3. 设置页数和产品数量
4. 点击开始搜索

## ⚙️ 推荐配置
- 小规模测试：20页，100产品
- 中等规模：50页，500产品  
- 大规模：100页，2000产品

## 🛡️ 稳定性保证
- 分批处理，每批20个产品
- 内存自动清理，不会溢出
- 错误自动恢复，单个失败不影响整体
