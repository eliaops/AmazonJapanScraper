## 🎮 用户操作指南

### 步骤1: 启动软件
```bash
# Windows用户
双击 run.bat

# macOS/Linux用户
python3 main.py
```

### 步骤2: 选择搜索方式

**方式A: 使用预设类目**
1. 在"选择商品类目"下拉菜单中选择"电脑/周边设备"
2. 系统会自动显示建议关键词：`コンピュータ, パソコン, PC`

**方式B: 使用自定义关键词**
1. 选择"自定义关键词"类目
2. 在"自定义关键词"输入框中输入日文关键词，如：
   - `ノートパソコン` (笔记本电脑)
   - `ゲーミングPC` (游戏PC)
   - `タブレット` (平板电脑)

### 步骤3: 设置搜索参数
- **最大页数**: 建议设置10-20页
- **最大产品数**: 设置500个（根据需要调整）

### 步骤4: 开始提取
1. 点击"开始提取"按钮
2. 软件会显示实时进度
3. 等待提取完成

### 步骤5: 导出数据
1. 提取完成后，点击"导出数据"
2. 选择Excel或CSV格式
3. 保存到指定位置
